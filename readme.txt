Подключение Вашего Интернет-магазина к онлайн-кассе и отправка фискальных чеков на e-mail покупателя при покупках через сайт

== Description ==

Данное решение позволяет подключить Ваш Интернет-магазин к облачному сервису МОБИКА, регистрировать и отправлять покупателю фискальные чеки на онлайн-кассе, соответствуюшие требованиям 54-ФЗ, при расчетах с использованием электронных средств платежей в сети Интернет.

### Возможности плагина

  - автоматическая фискализация платежей (формирование и отправка чека на e-mail покупателя);
  - оформление чека с зачетом предоплаты (при поступлении денег на счет) и чека полного расчета с зачетом аванса (при отгрузке товара или выполнении услуги). В полном соотвествии с правилами налогового законодательства, действующими с 1 июля 2019 года;
  - оформление чеков возврата.


### Описание работы

Пользователь в настройках плагина выбирает:

1. Cпособы оплаты (например, Яндекс.Касса, PayPal и Robokassa), при оплате через которые необходимо формировать фискальный чек.
2. Cтатус заказа в WooCommerce (рекомендуется статус "Обработка"), при котором будет формироваться чек прихода предоплаты.
3. Cтатус заказа в WooCommerce (рекомендуется статус "Выполнен"), при котором будет формироваться чек прихода с полным расчетом, включаещим зачет предоплаты в качестве аванса.
4. Как формировать чеки возрата прихода (при выборе соответсвующей опции чеки возварта будут формироваться автоматическ при изменении статуса заказа на "Возвращен"). 

При изменении статус заказа в WooCommerce плагин формирует и отправляется запрос на создание чека в облачный сервис "МОБИКА". Сервис отправляет команду на "Облако-1Ф", которая формирует чек. Новый чек в соотвествии с законом отправляется Оператору фискальных данных и на e-mail покупателя. 

Важно! 54-ФЗ обязует выдать электронный чек клиенту. Для того чтобы электронный чек был выслан клиенту на электронную почту, необходимо сделать обязательным поле e-mail  в форме оформления заказа.

### Сервис МОБИКА

После подключения к сервису Вам будут предоставлена касса "Облако-1Ф" (в собстенность или в аренду - на выбор) и индивидуальные ID и ключ для доступа к Вашей кассе. Эти параметры (ID и ключ) вводятся в настройках плагина (параметры "Группа устройств" и "Ключ API"). Подробности подключения к облачному кассовому сервису МОБИКА можно узнать на сайте www.mobika-online.ru.  

== Installation ==

**Также можно установить плагин через админ панель, используя форму поиска плагинов.**
1. Откройте вкладку Плагины (Plugins), кликните Добавить новый (Add New), в строке поиска введите название плагина (МОБИКА) и нажмите кнопку Enter на клавиатуре:
2. Выберите WooCommerce МОБИКА Онлайн-касса из результатов поиска и нажмите Установить
3. Когда плагин будет установлен, кликните ссылку Активировать плагин (Activate Plugin).
4. Настройте плагин в соответсвии с разделом "Описание работы". Параметры "Группа устройств" и "Ключ API" предоставлются менеджером  при подключени к сервису МОБИКА.
